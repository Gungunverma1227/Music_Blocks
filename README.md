# Music Blocks Composer

A browser-based drag and drop music composition tool built with 
HTML, CSS and Vanilla JavaScript.

## What it does
- Drag note, drum and volume blocks onto a canvas to compose music
- Press Play to hear your composition using Web Audio API
- Control BPM with a slider
- Supports pitch notes, kick, snare and hi-hat drum sounds
- Sequence panel shows your composition in order

## How to run
Just open `musicblocks.html` in any browser. No install needed.

## Built with
- HTML and CSS
- Vanilla JavaScript
- Web Audio API
